# AI Research Paper Summarizer


An AI-powered application that summarizes research papers and extracts important insights from PDF documents using Natural Language Processing (NLP).


## Features

- Upload research papers in PDF format
- Extract text from PDFs
- Generate concise summaries
- Simple and user-friendly interface
- NLP-based text processing


## Tech Stack

- Python
- Streamlit
- NLTK
- PyPDF2
- PyMuPDF
- Pillow



## Run the Project

python -m streamlit run app.py
